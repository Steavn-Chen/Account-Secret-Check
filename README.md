### 啓動方式
 
- 將專案clone到本地端

https://github.com/Steavn-Chen/Account-Secret-Check.git

- 進入到專案資料夾

cd Account-Secret-Check

- 安裝 npm

npm install

- 到終端機執行專案

npm run start

- 終端機出現以下字句，表示專案成功開啓。

Account-Secret-Check Web on http://localhost/3000


### 開發環境

- Node.js -10.15.0
- Express- 4.17.1
- Express-Handlebars-5.3.4
